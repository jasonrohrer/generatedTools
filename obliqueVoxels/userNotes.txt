All previously-listed issues are addressed (2026-07-12):

--DONE: Scribble Select tool (tool 6): left-drag paints every touched voxel
  into the selection; erase mode (E) un-paints.  Shares the selection with the
  marquee Select tool.

--DONE: The marquee Select tool now has "below" and "above" depth sliders.
  "below" sweeps the marquee down into the solid from the clicked surface;
  "above" sweeps it up out of the surface (both like the Box tool's thickness,
  but two-sided).

--DONE: Draggable separators between each side panel and the view.  Grab the
  thin handle at either boundary and drag.  The palette grid re-wraps to the new
  width, so wide ramps can be made to sit on one line.

--DONE: Flat and Quick-Light preview now pick a representative color from the
  MIDDLE of a voxel's ramp instead of its first index, and skip pure black when
  the ramp holds any non-black color -- so a ramp that starts on black no longer
  makes the whole voxel read as black.  (Natural render mode still uses the base
  color as the material, by design; switch to Palette-ramp mode to shade a black-
  based ramp across its range.)

--DONE: Lights have an "infinite (sun)" toggle.  A sun light casts parallel rays
  with no distance falloff (x,y,z is read as the direction toward the sun), so
  you get flat directional lighting instead of the local desk-lamp halo.  Saved
  in .ovox as an optional 7th field on the L line (older files still load).

--DONE: Real file pickers for Open / Save / Export PNG / Load palette.  They
  list the current directory (folders first), filter by the relevant extension,
  hide dotfiles (but keep ".." to go up), and let you click an entry or type a
  name.
